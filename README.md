# Smart-Heating

Currently, It's a Django web application that visualizes and analyzes temperatures from two thermal sensors via raspberry pi 2 in real-time.

## Hardware:
- Raspberry Pi 2
- Two thermal sensors DS18B20
- Jumper wires
- Breadboard
